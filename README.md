# Sistema de Gestión de Cuentas Bancarias

Este es un proyecto de un sistema simple de gestión de cuentas bancarias. Permite gestionar clientes y realizar operaciones como la búsqueda de clientes con base a su ID.

## Descripción

El proyecto permite la creación, consulta y gestión de clientes dentro de un sistema bancario. Se implementan servicios de backend utilizando Java, Spring Boot, y JPA. Este proyecto incluye pruebas unitarias, de integración, y configuraciones para garantizar que los servicios funcionen correctamente.

## Tecnologías Utilizadas

- **Java**: Lenguaje de programación principal.
- **Spring Boot**: Framework para la creación del backend.
- **Spring Data JPA**: Para la interacción con la base de datos.
- **H2**: Base de datos en memoria utilizada en las pruebas de integración.
- **JUnit 5**: Para la ejecución de pruebas unitarias y de integración.
- **Mockito**: Para la ejecución de pruebas unitarias.
- **Gradle**: Para la gestión de dependencias y ejecución de pruebas.
